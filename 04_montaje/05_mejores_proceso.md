# 🟢 MEJORA 2

📌 ¿Qué mejorarías?

Cambio de pasta térmica

📌 ¿Por qué?

Debido a que los ordenadores se usan alrededor de 12h diarias con programas medianamente estrictos como el packet tracer y visual studio code se puede llegar a sobrecalentar

📌 ¿Cómo lo harías mejor?

cambiarle la pasta térmica entre trimestres

📌 Beneficio:

mayor seguridad al equipo

# 🟢 MEJORA 3



📌 ¿Qué mejorarías?

Cambio de placa base 

📌 ¿Por qué?

Las placas base están bastante antiguadas y siguen usando conector VGA el cual lleva obsoleto bastantes años

📌 ¿Cómo lo harías mejor?

Comprando placas bases más modernas con conectores HDMI o DisplayPort

📌 Beneficio:

Mejor resolución de pantalla y mejor rendimiento
